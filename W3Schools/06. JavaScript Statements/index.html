Example of JavaScript statements.

let x, y, z; // Statement 1
x = 5; // Statement 1
y = 6; // Statement 1
z = x + y; // Statement 1

JavaScript Statements are Commands to the Browser

Pada HTML, Statemen dieksekusi di browser.

JavaScript Code is a Sequence of Statements

Program JavaScript adalah daftar statemen yang akan dieksekusi oleh komputer.

JavaScript Statements are Separated by Semicolons

Statemen JavaScript dipisah dengan tanda titik-koma ( ; ).

Multiple Statement on One Line is Allowed

Statement lebih dari satu dalam satu baris diperbolehkan, asalkan dipisah dengan tanda titik-koma ;

JavaScript Statements can be Grouped Together in Code Blocks

Statemen JavaScript bisa dikelompokan bersama didalam kode blok. Blok kode JavaScript ditulis diantara { ... }



You Can Break a Code Line After an Operator or a Comma

Tempat terbaik untuk break baris kode adalah setelah operator atau koma.
