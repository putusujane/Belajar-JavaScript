Dimana Meletakkan Kode JavaScript?

Dalam HTML, kode JavaScript diletakkan diantara tag script.

Apa itu Function dan Event Pada JavaScript?

function adalah blok code JavaScript, yang dapat dieksekusi saat "dipanggil". Misal: suatu fungsi dapat dipanggil ketika suatu event/peristiwa terjadi, seperti ketika pengguna menekan tombol.
