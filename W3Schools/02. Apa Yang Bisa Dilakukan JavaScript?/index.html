JavaScript Bisa Mengubah Konten HTML

Salah satu dari banyak metode digunakan adalah getElementById().

Contoh dibawah ini mencari elemen HTML dengan id "hello", dan mengubah konten elemen menjadi "Hello, JavaScript".

Hello, World!

Selamat datang!

Catatan!: JavaScript bisa menggunakan tanda kutip tunggal dan ganda.

JavaScript Dapat Mengubah Nilai Atribut HTML

JavaScript dapat mengubah nilai src atribut (Sumber) dari sebuah tag img.

JavaScript Dapat Mengubah Style CSS di HTML

Mengubah style elemen HTML dengan mengubah atribut CSS.

Ukuran teks ini menjadi besar, Warna menjadi merah.

JavaScript Dapat Menyembunyikan Elemen HTML

Dapat dilakukan dengan mengubah atribut display: none;

Teks ini akan hilang.

JavaScript Dapat Menampilkan Elemen HTML

Dapat dilakukan dengan mengubah atribut style display: block;.

Teks ini akan tampil.
