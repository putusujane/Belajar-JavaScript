Belajar JavaScript JSON Arrays

Contoh JSON string:

'["Ford", "BMW", "Fiat"]'

Didalam string JSON ada sebuah JSON array literal.

Array didalam JSON hampir sama sebagai array di JavaScript.

Pada JSON, nilai array harus bertipe string, number, object, array, boolean, atau null.
Dalam JavaScript, nilai array semua bisa seperti diatas, dan lainnya seperti
ekspresi, termasuk function, datem dan undefined.
