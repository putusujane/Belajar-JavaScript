Belajar JavaScript Object

Didalam JavaScript, Object adalah raja. Jika kita paham Object, maka kita paham JavaScript.

Dalam JavaScript, hampir "Semuanya" adalah Object. Contoh:

  • Boolean bisa menjadi Object (Jika diartikan dengan keyword new)
  • Number bisa menjadi Object (Jika diartikan dengan keyword new)
  • String bisa menjadi Object (Jika diartikan dengan keyword new)
  • Date selalu bertipe Object
  • Math selalu bertipe Object
  • Regular Expressions selalu bertipe Object
  • Array selalu bertipe Object
  • Function selalu bertipe Object
  • Object selalu bertipe Object

Semua nilai JavaScript, kecuali tipe data primitif, adalah Object.

JavaScript Primitives

Nilai primitif adalah nilai yang tidak mempunyai properti atau method.

3.14 adalah nilai primitif.

Tipe data primitif adalah data yang mempunyai nilai primitif.

Diketahui JavaScript memiliki 7 tipe dari tipe data primitif.

Contoh:

  • string
  • number
  • boolean
  • null
  • undefined
  • symbol
  • bigint

Immutable

Data primitif adalah immutable (Data yang keras, tidak bisa diubah).
Jika x = 3.14, kamu bisa mengubah nilai dari x, tetapi kamu tidak bisa mengubah nilai dari 3.14.

Value Type Comment
"Hello" string "Hello" selalu "Hello"
3.14 number 3.14 selalu 3.14
true boolean true selalu true
false boolean false selalu false
null null (Object) null selalu null
undefined undefined undefined selalu undefined
