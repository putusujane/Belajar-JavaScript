    • JavaScript Statement

    Program JavaScript adalah daftar statement yang akan dieksekusi oleh komputer.

    JavaScript Numbers

    Angka bisa ditulis dengan atau tanpa desimal.

    JavaScript Strings

    String bisa ditulis dengan tanda petik satu atau dua.

    JavaScript Variables

    Pada contoh ini, a disebut variabel. Kemudian, a assign dengan nilai 6.

    JavaScript Operators

    JavaScript menggunakan operator aritmatika untuk menghitung suatu nilai (seperti aljabar / algebra).

    JavaScript Assignment

    Didalam JavaScript, operator = digunakan mengassign-kan nilai ke variabel.

    JavaScript Expressions

    Ekspresi untuk menghitung nilai / value.

    JavaScript Keywords

    Kata kunci var untuk membuat variabel.

    JavaScript Comments

    Comment atau komentar tidak akan dieksekusi atau ditampilkan di web browser atau konsole.

    JavaScript Adalah Case Sensitive

    Coba untuk mengubah variabel lastName menjadi lastname.
