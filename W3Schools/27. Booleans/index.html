Belajar JavaScript Booleans

Data boolean pada JavaScript merupakan representasi satu dari dua nilai, yaitu: true atau false / benar atau salah.

Dalam pemrograman kita butuh tipe data yang hanya mempunyai satu dari dua nilai, seperti:

  • YA atau TIDAK
  • ON atau OFF
  • BENAR atau SALAH
