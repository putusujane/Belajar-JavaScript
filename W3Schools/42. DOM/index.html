JavaScript DOM (Document Object Model)

Dengan HTML DOM, JavaScript bisa mengakses dan mengubah semua elemen pada dokumen HTML.

HTML DOM (Document Object Model)

Saat halaman web ter-load, browser membuat DOM pada halaman.
Model HTML DOM terbuat dari cabang pada Objects.

Dengan model object, JavaScript mendapatkan tenaga untuk membuat HTML dinamis, seperti:

  • Mengubah semua elemen HTML
  • Mengubah semua atribut HTML
  • Mengubah semua style CSS pada halaman
  • Menghapus elemen dan atribut HTML yang sudah ada
  • Menambah elemen dan atribut baru pada HTML
  • Bereaksi pada semua even HTML yang sudah ada.
  • Membuat even baru pada halaman HTML.
