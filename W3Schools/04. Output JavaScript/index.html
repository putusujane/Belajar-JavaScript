    • Cara-cara Menampilkan Data Pada JavaScript di HTML

    Menampilkan data menggunakan JavaScript di HTML yaitu:

    • Menulis kedalam elemen HTML menggunakan innerHTML
    • Menulis kedalam output menggunakan document.write()
    • Menggunakan kotak alert (peringatan) menggunakan window.alert()
    • Menggunakan console browser, menggunakan console.log()

    Menulis Menggunakan innerHTML

    Untuk mengakses elemen HTML

    Menggunakan document.write()

    Untuk tujuan pengujian.

    Menggunakan window.alert()

    Untuk pengujian dan menampilkan di console browser.
